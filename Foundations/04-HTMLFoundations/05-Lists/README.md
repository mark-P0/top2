# Lists

**Unordered lists**, i.e. _bullet_ lists

- This is
- an unordered list
- in Markdown lmao

```html
<ul>
  <li>This is</li>
  <li>an unordered list</li>
  <li>in regular HTML</li>
</ul>
```

**Ordered lists**, i.e. _numbered_ lists

1. This is
2. an ordered list
3. in Markdown lol

```html
<ol>
  <li>This is</li>
  <li>an ordered list</li>
  <li>in regular HTML</li>
</ol>
```
